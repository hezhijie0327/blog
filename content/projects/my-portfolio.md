# 个人作品集网站

## 项目简介

这是我的个人作品集网站，展示了我的项目经历和技术博客。网站采用现代化设计，注重用户体验和性能优化。

## 主要特性

### 🎨 现代化设计
- 采用现代化的设计语言
- 参考优秀开源项目的设计理念
- 支持深色/浅色主题切换
- 完全响应式，适配各种设备

### ⚡ 性能优化
- 基于 Next.js 16 的静态生成
- 图片懒加载和优化
- 代码分割和按需加载
- 极致的首屏加载速度

### 🛠️ 技术特点
- **框架**: Next.js 16 + React 19
- **语言**: TypeScript
- **样式**: Tailwind CSS v4
- **内容**: Markdown + MDX
- **动画**: Framer Motion
- **部署**: GitHub Pages

### 📱 移动端优化
- 移动优先的设计理念
- 流畅的触摸交互
- 优化的移动端导航
- 适配各种屏幕尺寸

## 项目结构

```
src/
├── app/                    # Next.js App Router
│   ├── blogs/             # 博客页面
│   ├── projects/          # 项目页面
│   └── layout.tsx         # 根布局
├── components/            # React 组件
│   ├── ui/               # 基础 UI 组件
│   ├── Navigation.tsx    # 导航组件
│   └── GitHubComments.tsx # 评论组件
└── lib/                  # 工具函数
    └── content.ts        # 内容管理
```

## 技术亮点

### 1. 组件化架构
使用现代化的组件化设计，提高代码复用性和维护性。

### 2. 类型安全
全面使用 TypeScript，确保代码的类型安全和开发体验。

### 3. 现代化 UI
基于 Tailwind CSS v4 和 Framer Motion，打造流畅的用户界面。

### 4. 内容管理
基于 Markdown 的内容管理系统，支持代码高亮和数学公式。

### 5. SEO 优化
完善的 SEO 配置，提升搜索引擎排名。
